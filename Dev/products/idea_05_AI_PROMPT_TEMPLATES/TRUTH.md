# Statement of Truth - AI Prompt Templates Marketplace

## Core Purpose
Provide a curated library of high-quality, battle-tested AI prompts for common business and creative tasks, saving users hours of prompt engineering trial-and-error.

## Target User
- Content creators using ChatGPT/Claude daily
- Small business owners automating tasks with AI
- Marketers creating campaigns
- Developers building AI features
- Consultants delivering client work

## Value Proposition
Buy proven prompts ($5-$15 each) instead of spending 2-4 hours testing variations. Get results immediately with templates that actually work.

## MVP Features (v1.0 ONLY)
- [ ] **Prompt Library**
  - 50 curated prompts at launch
  - Categories: Marketing, Content, Code, Business, Creative
  - Each prompt includes:
    - Title & description
    - Full prompt text (copy-paste ready)
    - Example output
    - Use case scenarios
    - Variables to customize (e.g., [PRODUCT_NAME], [TARGET_AUDIENCE])

- [ ] **Browse & Search**
  - Category filters
  - Search by keyword
  - Sort by popularity, newest, price

- [ ] **Purchase & Access**
  - Stripe payment (one-time per prompt)
  - Instant access after purchase
  - Download as TXT file
  - Copy to clipboard

- [ ] **User Dashboard**
  - View purchased prompts
  - Access prompt library
  - Re-download anytime

- [ ] **Bundle Offers**
  - "Marketing Pack" (10 prompts for $49)
  - "Content Creator Pack" (15 prompts for $69)
  - "Business Automation Pack" (12 prompts for $59)

## Anti-Scope (DO NOT BUILD)
- Custom prompt creation service (keep it productized)
- Prompt builder/generator tool (just sell templates)
- Community submissions (curate ourselves for v1)
- Multi-language prompts (English only v1)
- Subscription model (one-time purchases only)

## Tech Stack
- Frontend: Next.js (React)
- Auth: NextAuth or Clerk
- Database: Supabase (prompts, users, purchases)
- Payments: Stripe (one-time payments)
- Hosting: Vercel

## Monetization
- Model: One-time purchases
