# Tweet Thread Generator
**Category:** Content Creation
**Price:** $(System.Collections.Hashtable.Price)
**Description:** Turn articles or ideas into viral Twitter threads.

## Full Prompt
Convert the following text into a 10-tweet thread: [INSERT_TEXT]. The first tweet must be a strong hook. The last tweet must be a recap and CTA. Number the tweets 1/x.

## Example Output
1/10 This is the hook.

2/10 Point one.

...

10/10 Recap. Follow for more!

## Use Cases
- Repurposing content
- Building Twitter audience
- Sharing news
