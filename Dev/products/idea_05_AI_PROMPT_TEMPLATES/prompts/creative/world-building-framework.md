# World Building Framework
**Category:** Creative Writing
**Price:** $(System.Collections.Hashtable.Price)
**Description:** Flesh out fictional worlds.

## Full Prompt
Help me build a fantasy world. Magic system based on: [IDEA]. Political structure: [TYPE]. Geography: [MAIN_FEATURE]. Describe the daily life of a commoner.

## Example Output
The world of X is ruled by mages. Commoners must pay a magic tax.

## Use Cases
- Fantasy novels
- D&D campaigns
- Game lore
