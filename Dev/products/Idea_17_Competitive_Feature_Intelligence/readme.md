# LaunchRadar - Competitive Feature Intelligence

> **Status:** Scaffolding Complete 🏗️  
> **Mission:** Know when competitors ship before your customers tell you.

## 🎯 Problem
Competitors ship features and change pricing constantly. Product teams monitor manually or find out too late.

## 💡 Solution
Automated monitoring of competitor sites with AI-powered change summarization.

## 📊 Core Features (MVP)
- Competitor watchlist
- Change detection (features, pricing, positioning)
- AI summaries (not raw diffs)
- Slack/email digests
- Feature comparison matrix
