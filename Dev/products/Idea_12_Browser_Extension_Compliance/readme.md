# ExtensionAudit - Browser Extension Compliance Scanner

> **Status:** Scaffolding Complete 🏗️  
> **Mission:** Give security teams visibility into browser extensions across the org.

## 🎯 Problem
Employees install browser extensions freely. IT has no visibility into which extensions are running or what permissions they've been granted.

## 💡 Solution
Lightweight agent that inventories extensions, risk-scores by permissions, and enforces policies.

## 📊 Core Features (MVP)
- Chrome extension inventory agent
- Permission risk scoring
- Known malicious extension database
- Policy enforcement (allowlist/blocklist)
- Admin dashboard
