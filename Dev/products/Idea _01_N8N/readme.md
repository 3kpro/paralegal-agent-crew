A curated collection of battle-tested n8n workflows designed to save 10-50 hours of development time per implementation.

### 🎥 [Content Creator Pack](workflows/youtube_to_blog.json)

- **Features:** YouTube Video Monitoring -> Transcription -> AI Blog Writing -> WordPress Draft.
- **Status:** ready for testing.

### 📧 [Small Business Pack](workflows/lead_capture.json)
Instant Lead Response system.
- **Features:** Web Form Webhook -> CRM Entry (HubSpot) -> Personalized Welcome Email.
- **Status:** ready for testing.

### 🗓️ [Social Media Scheduler Pack](workflows/social_scheduler.json)
The 'Set and Forget' social manager.
- **Features:** Airtable Schedule -> Buffer Posting -> Status Updates.
- **Status:** ready for testing.

## 🚀 Getting Started
1. **Read the Manual**: Check out [docs/WORKFLOW_MANUAL.md](docs/WORKFLOW_MANUAL.md) for setup instructions.
2. **Import**: Load the `.json` files directly into your n8n instance.
3. **Configure**: Add your API keys as prompted by the nodes.

## 📂 Project Structure
- `/workflows`: The raw JSON files for import.
- `/docs`: Detailed guides and setups.
- `TRUTH.md`: The guiding vision and product specs.
