# AgentMeter - AI Agent Cost Optimizer

> **Status:** Scaffolding Complete 🏗️  
> **Mission:** Know exactly where your AI budget is going.

## 🎯 Problem
Companies deploying AI agents get monthly bills with zero attribution. No visibility into per-conversation costs or which agents are wasteful.

## 💡 Solution
Instrumentation SDK + real-time cost dashboard with budget guardrails and waste detection.

## 📊 Core Features (MVP)
- SDK wrappers for OpenAI/Anthropic
- Real-time cost dashboard
- Waste detection
- Budget guardrails
- Optimization recommendations
