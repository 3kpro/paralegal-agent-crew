# ReviewLens - Code Review Bias Detector

> **Status:** Scaffolding Complete 🏗️  
> **Mission:** Surface hidden patterns and bias in your code review process.

## 🎯 Problem
Code reviews are subjective. Some reviewers nitpick juniors while rubber-stamping seniors. Teams have no visibility into these patterns.

## 💡 Solution
Analyze PR review history to detect statistical anomalies in how different team members get reviewed.

## 📊 Core Features (MVP)
- GitHub/GitLab integration
- Reviewer pattern analysis
- Bias detection dashboard
- AI comment quality scoring
- Anonymized benchmarking
