# PostmortemAI - Incident Postmortem Generator

> **Status:** Scaffolding Complete 🏗️  
> **Mission:** Make postmortems write themselves.

## 🎯 Problem
Postmortems are dreaded busywork. They get written late (if at all), lack structure, and action items disappear.

## 💡 Solution
Auto-generate postmortems from Slack threads and PagerDuty alerts using AI to construct the narrative.

## 📊 Core Features (MVP)
- Slack thread ingestion
- PagerDuty alert timeline
- AI narrative generation
- Action item extraction
- Export to Notion/Confluence
