# SchemaSentry - Schema Drift Detector

> **Status:** Scaffolding Complete 🏗️  
> **Mission:** Catch schema changes before they break your pipelines.

## 🎯 Problem
Data teams build pipelines assuming upstream schemas stay stable. They don't. Teams discover schema changes when dashboards break.

## 💡 Solution
Monitor data sources for schema changes and alert teams with severity classification and impact analysis.

## 📊 Core Features (MVP)
- Schema snapshot registry
- Change detection (add/remove/type)
- Severity classification
- Impact analysis
- Slack/email alerts
