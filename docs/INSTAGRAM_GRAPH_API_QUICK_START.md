# Instagram Graph API - Quick Start for 3KPRO.AI

**Correct API**: Instagram Graph API (NOT Basic Display API)  
**Link**: https://developers.facebook.com/docs/instagram-api

---

## What You Need

✅ Instagram Business Account (you have this!)  
⏳ Facebook Page (need to create)  
⏳ Meta Developer App  
⏳ API Access Token  

---

## Step-by-Step

### 1. Create Facebook Page (Required)
Even though your Instagram Business works without Facebook, the API requires it.

1. Go to: https://www.facebook.com/pages/create
2. Page Name: "3KPRO.AI" or "3K Pro Services"
3. Category: Technology / Marketing
4. Publish (takes 2 minutes)

### 2. Link Instagram to Facebook Page
1. Instagram App → Settings → Account → Sharing to other apps
2. Facebook → Connect Account
3. Select your 3KPRO.AI Facebook Page
4. Confirm the connection

### 3. Create Meta Developer App
1. Go to: https://developers.facebook.com/apps
2. Create App → Business
3. App Name: "3KPRO Content Automation"
4. Add Product → Instagram → Instagram Graph API

### 4. Get Access Token
Follow docs: https://developers.facebook.com/docs/instagram-api/getting-started

---

## What You Can Do With Instagram Graph API

✅ Publish posts (photos + captions)
✅ Schedule posts in advance
✅ Reply to comments
✅ Get post insights/analytics
✅ Search hashtags
✅ Manage multiple accounts

**Rate Limits**:
- 25 API-published posts per Instagram Business account per 24 hours
- Rate limit only applies to API posts (you can still post unlimited via Instagram app)

**Post Types Supported**:
- ✅ Single photo posts
- ✅ Single video posts
- ❌ Stories (not available via API)
- ❌ Reels (limited support)
