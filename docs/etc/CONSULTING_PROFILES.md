## DISCOVERY CALL SCRIPT

**Goals:**
1. Understand their problem
2. Qualify if it's automatable
3. Scope the work
4. Close or send proposal

**Questions to ask:**

**1. Problem Understanding:**
- "Walk me through the manual process you're doing now"
- "How much time does this take per week?"
- "What's the most frustrating part?"
- "What happens if this doesn't get done?"

**2. Technical Discovery:**
- "What tools are you currently using?"
- "Do you have API access to those tools?"
- "Any existing automations in place?"

**3. Budget & Timeline:**
- "What's your timeline for getting this solved?"
- "Have you budgeted for automation?"
- "What would success look like 30 days from now?"

**4. Close:**
- "Based on what you've told me, I can definitely automate this"
- "It would take [X days] to build and cost around [Y]"
- "Does that work for you?"

**If yes:**
- "Great! I'll send you a detailed proposal by [tomorrow/end of day]"
- "Anything else I should include in the proposal?"

**If maybe:**
- "No problem. What would help you make a decision?"
- "Would a smaller pilot project help you see the value?"
