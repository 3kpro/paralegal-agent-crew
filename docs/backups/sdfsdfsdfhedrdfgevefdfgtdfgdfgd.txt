The Technical Breakdown (For Due Diligence)
This project is a masterclass in leveraging a modern, scalable, and secure tech stack for rapid development without sacrificing quality.

Frontend: Next.js 15 (App Router). We chose this for its performance benefits—Server Components, optimized data fetching, and best-in-class SEO. The UI is built with Tailwind CSS for utility-first efficiency and Framer Motion for fluid, professional animations that create a premium user experience.

Backend & Database: Supabase. This isn't just a database; it's our entire backend-as-a-service. We leverage its managed PostgreSQL for data, its robust Auth system with Row Level Security (isolating user data at the database level), and its high-performance RPC Functions to handle business logic like usage limits efficiently. This stack allows us to build secure, complex features incredibly fast.

AI & ML Architecture:

Multi-Provider Integration: We integrate directly with AI providers like Google, OpenAI, and Anthropic. This reduces latency and vendor lock-in compared to using middlemen like OpenRouter.
Serverless ML Deployment: The Viral Score™ ML model is designed to be served via Google Cloud Run. This is a world-class serverless architecture that scales to zero, meaning our ML inference costs are near-$0 at launch and scale perfectly with demand. It's the most cost-effective way to deploy a production-grade model.
Data Security: User API keys are never stored in plaintext. They are encrypted in the database using AES-256-GCM, a modern, authenticated encryption standard, via our custom lib/encryption.ts module.
Deployment & Performance:

Vercel: The perfect companion to Next.js, providing seamless CI/CD, automatic scaling, and global distribution via its Edge Network for low-latency access worldwide.
API Optimization: Our APIs are built for speed. We use Promise.all to parallelize independent operations and a "fire-and-forget" pattern for non-blocking analytics, ensuring the user-facing response is as fast as possible.
This isn't a prototype; it's a production-ready, scalable, and secure platform built with the best tools available today.
