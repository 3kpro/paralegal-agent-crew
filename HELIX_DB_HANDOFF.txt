# Supabase AI Assistant Handoff - Helix Implementation

## Objective
Apply the new database schema for the **Helix AI Assistant**. This schema supports the chat history, brand DNA profiling, and knowledge base (RAG) features.

## Migration Files
1.  **`supabase/migrations/20251126000000_init_helix_ai.sql`**
    *   Creates base tables: `helix_brand_dna`, `helix_knowledge_base`, `helix_sessions`, `helix_messages`.
    *   Sets up RLS policies.
2.  **`supabase/migrations/20251126000001_add_helix_vectors.sql`**
    *   Enables `vector` extension.
    *   Adds `embedding` column (768 dimensions) to `helix_knowledge_base`.
    *   Creates HNSW index for fast cosine similarity search.
    *   Adds `match_knowledge_base` function for RAG.

## Action Required
Please run the following command to apply the migrations to the remote Supabase instance:

```bash
npx supabase db push
```

## Verification
After pushing, verify:
1.  `helix_brand_dna` table exists.
2.  `helix_knowledge_base` table has an `embedding` column.
3.  `vector` extension is enabled.
